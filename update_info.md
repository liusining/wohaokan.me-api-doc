# 更新个人信息

更新用户的昵称和个人简介。



### HTTP

> POST /update_info



### 参数

| 名称        | 类型   | 必须？ | 默认值 | 说明                 |
| ----------- | ------ | ------ | ------ | -------------------- |
| name        | string | 是     | 无     | 用户的昵称，不可为空 |
| age         | integer | 否    | 无     | 覆盖照片检测出的年龄 |
| description | string | 否     | `""`   | 用户的自我介绍       |



### 返回

**成功**

```json
{
    "status": 200,
    "msg": "OK",
    "result": {}
}
```



**失败**

```json
{
    "status": 400,
    "msg": "姓名不能为空",
    "result": {}
}
```
